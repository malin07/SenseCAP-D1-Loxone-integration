# SenseCAP-D1-Loxone-integration
Dieses Repository liefert eine vollständige Anleitung und die Code-Snippets, um ein SenseCAP D1 Touch-Display (geflasht mit OpenHASP) nahtlos in ein Loxone Smart Home zu integrieren. Die Kommunikation läuft lokal und blitzschnell über einen LoxBerry (MQTT Gateway).  Das perfekte, hochgradig individualisierbare und kostengünstige Smart-Home-Display!
